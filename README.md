## IT1214-Student-Grade-System

## Description
This is a Java console-based application developed to manage student academic records digitally.
The system allows teachers to add student details, view all students, search students by ID,
and calculate the average marks of all students.

## Features
* Add a new student
* Display all student records
* Search a student by Student ID
* Calculate average marks of all students

## Technologies Used
* Java
* Object-Oriented Programming (OOP)
* GitHub Version Control

## Student Class Attributes
* Student ID
* Student Name
* Marks

## How to Run
1. Open the project in a Java IDE (Eclipse, IntelliJ IDEA, VS Code, etc.).
2. Compile and run the Main class.
3. Use the menu options to manage student records.
